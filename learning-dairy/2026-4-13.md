## 一、迭代器
在STL很多数据结构里，内存并不是和数组一样是连续的。所以用原生的指针语法糖是没法遍历访问、随机访问的。于是出现了迭代器这个工具。**迭代器实际上就是指向元素的指针，但是比较智能，增加和减小代表了元素的下一个/上一个。**<br>
很多STL的方法是必须要使用迭代器的。
```cpp
#include<set>
set<int> st;
st.insert(1); st.insert(2); st.insert(3);
for(auto it = st.begin();it != st.end();it++) cout << *it << ' ';
st.erase(st.begin()+2);
// begin和end对应的是头迭代器和尾迭代器。
// 还有逆序迭代器rbegin()和rend()
// 下面是几个对迭代器操作的方法
advance(it,n);// 将迭代器it前移/后退n个位置，直接修改it，无返回值
next(it,n); prev(it,n);
// 不改变原迭代器，向后/向前移动迭代器。返回移动后的迭代器
```
注意迭代器其实是分类型的，数组和字符串的迭代器很自由，和普通指针类似，可以更改值。但是set和map的迭代器就只能用来查看元素和遍历，不能修改。<br><br>

## 二、STL数据结构
### <span id='vec'>1. vector</span>
一种时间效率不如原生数组的数组，通常用来存放一些结构体或者用来减小空间占用。
```cpp
#include<vector>
// 初始化是最重要的
vector<int> vec(n,0); // 大小为n，数据为0；
// 可以用迭代器拷贝别的STL的元素
vector<int> vec(st.begin(),st.end());
// 增删改查中需要注意的是insert
vector<int> xx;
vec.insert(vec.begin()+3,xx.begin(),xx.end());
// 参数全是迭代器：在第一个参数处插入后面两个参数所构成的区间。如果后面只有一个参数，插入这个参数(int类型迭代器/直接是int)
// vec.erase()方法和insert很像，不再赘述
```

### <span id = 'sq'>3. stack && queue && priority_queue</span>
三种非常重要但是非常简单的数据结构，至于为什么简单: 1.只有一种声明方法 2. 容器无法遍历 3.内部元素无法访问 4.连迭代器都没有。<br>
特点：stack后进先出，queue先进先出，priority_queue自动排序只能取top
```cpp
#include<stack>
#include<queue>
stack<int> stk;
stk.push(1);stk.push(2);stk.push(3);
stk.top(); // 3
stk.pop(); stk.top(); // 2

queue<int> q;
q.push(1);q.push(2);
q.front(); // 1
q.pop(); q.front(); 2

priority_queue<int> prq;
prq.push(1);prq.push(3);prq.push(2);
prq.top(); // 3
// 注意优先队列虽然有队列两个字，但是取元素用top而不是front
// 注意优先队列的排序函数，应该这么写，如下！！
priority_queue<int,vector<int>,greater<int>> prq; // 这样就由小到大排序
```
这里如果要补充的话就补充一下比较函数吧。事实上priority_queue可以自定义“比较函数”的，是一个返回类型为bool的函数。
```cpp
bool cmp(int a,int b){
    if(a%10 != b%10) return a%10>b%10;
    else return a>b;
} // 这个函数表示，先比较a和b的个位谁大，如果相同再比较谁更大。
priority_queue<int,vector<int>,cmp> prq;
// 这里cmp函数返回的是true/false，在优先队列里会自动用这个函数取两个待比较的值传入进去，如果返回true，就不改变顺序，返回false就改变顺序

// 同理还可以放cmp的有map，set，sort
set<int,cmp>st;
map<int,string,cmp>mp;
sort(vec.begin(),vec.end(),cmp());
// 注意sort函数里面cmp要加括号！！！记住即可
```
stack，queue，priority_queue在题目中很常见，但是只是最简单的一环，按需取用即可。
### <span id='sm'>4.set && map</span>
set和map在题目里更常见，在leetcode简单和部分中等题里是大杀器。
set和map都分为很多不同的实现，如：set, unordered_set, multiset, unordered_multiset。map也是这样。具体的区别如下
```cpp
// set 和 map 的底层是红黑树实现，自动维护顺序，自动去重
// unordered_set 和 unordered_map 是哈希表，只有去重
// multiset multimap排序 不去重
// unordered_multiset unordered_multimap不排序不去重
map<int,int>mp;
mp[1]=1; // 直接这么写就可以添加(1,1)的键值对
```
！！！重点要注意，对于map类的数据结构，**键是最重要的**，而不是值！！！

## <span id='algo'>三、Algorithm函数</span>
常用的只有四个
```cpp
#include<algorithm>
sort(//迭代器1,//迭代器2,//排序函数)
vector<int> arr;
sort(arr.rbegin(),arr.rend(),cmp());
// 默认由小到大，字符串默认字典序

reverse(arr.begin(),arr.end()); // 反转

// 下面两个是二分查找函数，返回值是迭代器。要求原数组必须严格升序
lower_bound(arr.begin(),arr.end(),5);
// 找arr数组里面第一个>=5的数字
upper_bound(arr.begin(),arr.end(),5);
// 找arr数组里面第一个严格>5的数字
// 也可以自己传入比较器，但是必须保证数组按照这个比较器严格升序。

swap(vec1,vec2);
swap(a,b);
// swap用来交换两个数据类型相同的对象，也可以放解引用的迭代器

unique(vec.begin(),vec.end());
去掉相邻的重复元素。如果原数组有序，能做到去重的效果。
```
